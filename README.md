# Status Network OS

Old computer style gateway to Status Network ecosystem

![demo](.github/assets/demo.png)

## Installing

1. Install dependencies
   ```bash
   pnpm install
   ```

2. Fire up the server and watch files

   ```bash
   pnpm dev
   ```
